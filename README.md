# Velo-O-Wisp

# ![Velo-O-Wisp](preview-1.png)

Velo-O-Wisp — учебный веб-проект для автоматизированного учета велосипедов на парковках у станций метро с использованием технологий искусственного интеллекта.

Проект разработан в рамках производственной практики по дисциплине **«Практика (Системы искусственного интеллекта)»** и представляет собой информационную систему компьютерного зрения, позволяющую автоматически обнаруживать велосипеды на фотографиях, подсчитывать их количество и сохранять результаты анализа.

## Основные возможности

* Загрузка изображения с компьютера
* Автоматическое обнаружение велосипедов с помощью YOLOv8
* Подсчет количества найденных велосипедов
* Отображение изображения с Bounding Box
* Сохранение истории анализов в Supabase
* Просмотр истории выполненных анализов
* Экспорт истории результатов
* Современный адаптивный интерфейс

## Структура базы данных

Проект использует таблицу:

* history — история выполненных анализов

Структура таблицы:

* id
* file_name
* upload_date
* bicycle_count
* processing_time
* original_image
* result_image

## Запуск проекта

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запустить сервер:

```bash
python app.py
```

или воспользоваться файлом

```text
run.bat
```

После запуска автоматически откроется сайт по адресу:

```
http://127.0.0.1:5000
```

## Принцип работы

1. Пользователь выбирает изображение с компьютера.
2. Изображение отправляется на сервер.
3. YOLOv8 обнаруживает объекты класса **bicycle**.
4. Выполняется подсчет найденных велосипедов.
5. На изображении автоматически отображаются Bounding Box вокруг найденных объектов.
6. Результат анализа сохраняется в базе данных Supabase.
7. Пользователь получает обработанное изображение и информацию о количестве найденных велосипедов.
